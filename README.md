## How to build

```bash
yarn build # Yarn
npm run build # NPM
```

## How to flatten my contract

It's so easy with waffle, you can just run 

```bash
npx waffle flatten
```

to get the flattened version of your contract.

Flattened contract will be in `./flatten` folder
